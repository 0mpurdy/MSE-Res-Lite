Reading the Volumes

To read a volume, choose one of the options from the 'Books' menu and then click one of the volume links on the subsequent author's page (e.g. Volume 1). This will take you to page one of the selected volume.

All Scripture references are underlined (hyperlinks), and clicking the reference will take you to the appropriate chapter/verse in the Bible (J.N.D. and Authorised alongisde each other).

You may also go directly to a book of the Bible by choosing the 'Bible' option on the 'Books' menu and then clicking a particular book.

Clicking the "go to synopsis" link at the start of a chapter in the Bible will take you to the relevant page of J.N.D.'s Synopsis.

Searching the Volumes

Choose what to search for

Type the words you wish to search for in the text box. The search engine is case insensitive i.e. searching for 'heavenly' will find all occurrences of 'heavenly', 'Heavenly' or 'HEAVENLY'. Any punctuation that you enter will be ignored (except for possessive apostrophes & hyphens).

Select the type of search from the 'Search Type' menu

  • Exact Match - will find sentences (or verses if searching the Bible) where the words occur consecutively, in the order in which you have typed them.
  • Sentence contains words in order - will find sentences (or verses if searching the Bible) in which all of the words occur in the order you typed them although not necessarily consecutively.
  • Sentence contains words - will find sentences (or verses if searching the Bible) in which all of the words occur in any order and not necessarily consecutively.
  • Paragraph contains words - will find paragraphs (or chapters if searching the Bible) in which all of the words occur in any order and not necessarily consecutively.

Choose authors to search

From the rows of check boxes choose whether you wish to search the Bible, Hymns or volumes of ministry. You can search more than one of them at once. Clicking the 'All' button (when one or more of the check boxes are selected) will check all the boxes, clicking 'All' again (when all the check boxes are selected) will uncheck all the boxes.

Click search

Finally, click the 'Search' button and then, after a short time (depending on the speed of your computer), the search results page will be displayed. The first search of a particular author will be slower than subsequent searches for the same author since the index needs to be loaded in the first search.

Choose your result

Clicking on one of the references on the search results page will take you to that reference. The matched sentence/paragraph will be displayed at the top of your browser window.

Wildcards

If you search for 'delight*' then the search engine will find all sentences that contain one or more of the following words: delight, delights, delightful, delighted and delighting. Likewise, if you search for '*boat' then the search engine will find all sentences that contain one or more of the following words: life-boat, boat, lifeboat, pleasure-boat and church-boat. Finally, if you search for '*wait*' then the search engine will find all sentences that contain one or more of the following words: wait, waited, waiting, waits, awaits, waiteth, awaiting, await, awaited and waiter.

Wildcard searches take considerably longer than normal searches and the type of search selected in the 'Search Type' menu will be ignored.

Refining searches

When you have completed at least one search, the Refine button will be enabled. This allows you to search the results of the previous search.

Refine including

To refine the results to include a search term, type the search term and click refine
For example, search for the phrase "corn of wheat", close the results window, enter "contact with men" in the search string field and then click the Refine button. A reference in JBS volume 7 will be found.

Refine excluding

To refine the results to exclude a search term, prefix the search term you wish to exclude with an '!'
For example for the phrase "corn of wheat" again but this time enter "!die" in the search string field before clicking the Refine button. This time the search engine will find all sentences that contain "corn of wheat", but don't contain "die".

The results of a refined search can be further refined.

Very Frequent Words

There are a number of words (about 200) that occur so frequently (more than 1000 times) in ministry that they have not been indexed (for example: 'a', 'all', 'am', 'and', 'any' and 'are'). If these are the only words that you enter in the search string then the search engine will return no matches.

Options

To modify the default behaviour of the search engine, choose "Advanced" from the menu then select "View Config file". The configuration file will be displayed.

Updating the Ministry Search Engine

To update the Ministry Search Engine:

  • Open the Dropbox folder here.
  • Download and unzip the new version (more detailed instructions on Dropbox)
  • (Optional) Delete the older version of the MSE

Uninstalling the Ministry Search Engine

To remove the Ministry Search Engine:

  • Navigate to the folder where you store the ministry search engine.
  • Delete the files you wish to remove (mse.exe is the program, res folder is the files)
